# 📝 To-Do List (Only JS)

A simple command-line style To-Do app built using vanilla JavaScript.

## 🚀 Features

- Add, list, and delete tasks
- Runs using `prompt()` and `console.log()`

## ⚠️ Reality Check

- No UI (console-based)
- No data persistence
- Beginner-level project

## 🎯 Purpose

Built to practice JavaScript basics like loops, arrays, and conditionals.
